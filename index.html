Js Array

Types
  1. Push

    2. Adds new elements to the end of an array, and returns the new length

  2. Unshift

    3. Adds new elements to the beginning of an array, and returns the new length

  3. Pop

    4. Removes the last element of an array, and returns that element

  4. Shift

    5. Removes the first element of an array, and returns that element

  5. Slice

    6. Selects a part of an array, and returns the new array

  6. Splice

    7. Adds/Removes elements from an array

  7. toString

    8. Converts an array to a string, and returns the result

  8. Sort

    9. Sorts the elements of an array {alphabet,ascending,decending orders}

  9. IndexOf

    10. Search the array for an element and returns its position

  10. LastIndexOf

    11. Search the array for an element, starting at the end, and returns its position

Number to Integer
There are 4 common methods to round a number to an integer:
  1. Math.round(x) - Returns x rounded to its nearest integer
  2. Math.ceil(x) - Returns x rounded up to its nearest integer
  3. Math.floor(x) - Returns x rounded down to its nearest integer
  4. Math.trunc(x) - Returns the integer part of x (new in ES6)
